This scenario shows how to use a resource file to localize labels and tooltips for AppBar commands. Strings are placed into a separate string table (resources.resjson) so that they can be maintained and translated separately.

To see new resources, change the language by doing the following:

  • Go to the Desktop Control Panel > Language > Add a language that is supported by this sample > Move new language to the top of the list.
  • Languages supported: English (United States), French (France), German (Germany)
